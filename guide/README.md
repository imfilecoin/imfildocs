# ImFil钱包介绍

ImFil是一套同于FileCoin资产管理的钱包，支持普通钱包，多签钱包以及存储提供者钱包。

## 他是如何工作的

ImFil包含热钱包和冷钱包两个软件。热钱包用于线上交易，冷钱包离线使用，用于存储私钥，对交易签名。
